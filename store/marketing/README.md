## 🚀 Quick start — 3 platformy

### TikTok / Reels (najwyższy ROI dla fintech)

1. Wgraj `kredytai-ad-10s-9x16.mp4`
2. Caption z `MARKETING_COPY.md` → sekcja „Wersja krótka"
3. Hashtagi: `#kredyt #SKD #frankowicze #AI #fintech #Polska`
4. CTA: link w bio do landing (https://jakatora.github.io/kredyt-ai-site/)

### Facebook / Instagram Ads (paid)

1. Wgraj `facebook-post.png` (single image) lub `instagram-square.png` (carousel)
2. Caption z sekcji „Wersja Facebook"
3. Targetowanie z sekcji „Targeting"
4. Budget: 50-100 PLN/dzień na start, optymalizacja po 7 dniach

### YouTube wideo

1. Thumbnail: `youtube-thumbnail.png`
2. Wideo: użyj `kredytai-ad-10s-9x16.mp4` jako intro (przekonwertuj na 16:9 jeśli potrzeba) albo nakręć longer-form
3. Title: „Sankcja kredytu darmowego — sprawdź czy bank Cię oszukał (AI)"
4. Description z linkami do app + landing

## 📊 Tracking

Dla każdej kampanii ustaw UTM:
- `?utm_source=instagram&utm_medium=story&utm_campaign=launch2026`
- `?utm_source=facebook&utm_medium=feed&utm_campaign=launch2026`
- `?utm_source=tiktok&utm_medium=video&utm_campaign=launch2026`
- `?utm_source=youtube&utm_medium=thumbnail&utm_campaign=launch2026`

Backend Railway loguje URL → możemy mierzyć install-to-paid conversion per kanał.

## 🔁 Regeneracja

Jeśli chcesz inne warianty:
- Wideo Kling: kosztuje ~$1 za 10s pro, kolejne renderowanie ~5-8 min
- Canva: bezpłatnie, wybór z 4 wariantów

## ⚖️ Compliance

Wszystkie reklamy MUSZĄ mieć:
- Cena widoczna **49 zł** (transparency)
- Disclaimer „AI-asystent, nie zastępuje porady prawnej" przy obietnicach wyników
- Brak gwarancji konkretnej kwoty zwrotu („do 18 000 zł" jest OK, „dostaniesz 18 000 zł" — NIE)
- Brak twierdzenia że KredytAI to kancelaria prawna
